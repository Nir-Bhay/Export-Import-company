# Naaz Scrap – Industrial Export & Import Company Website

## Project Title
Naaz Scrap (formerly GM International/Enterprise)

## Tech Stack
- **Core**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS (via CDN)
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Inter, Poppins)
- **Build**: Static site – no bundler required

## Installation & Usage
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Nir-Bhay/Export-Import-company.git
   cd Export-Import-company
   ```
2. **Run locally** (choose one):
   - **VS Code Live Server**: Open `index.html` → right‑click → "Open with Live Server".
   - **Python HTTP server**:
     ```bash
     python3 -m http.server 8000
     open http://localhost:8000
     ```
   - **Directly**: Double‑click `index.html` (some browsers may restrict local JS features).
3. **Deploy**: Upload the files to any static‑hosting provider (GitHub Pages, Netlify, Vercel, etc.).
